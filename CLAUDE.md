# Global Rules — Apply in Every Workspace

## Tech Stack
- **npm** (package_manager)
- **TypeScript** 5.2.0 (language)
- **Node.js** (runtime)
- **Vitest** 0.34.0 (test_tool)
- **tsup** 8.0.0 (build_tool)

## Commands
- **build**: `npm run build`
- **build:watch**: `npm run build:watch`
- **clean**: `npm run clean`
- **rebuild**: `npm run rebuild`
- **verify**: `npm run verify`
- **lint**: `npm run lint`
- **lint:fix**: `npm run lint:fix`
- **format**: `npm run format`
- **typecheck**: `npm run typecheck`
- **test**: `npm run test`
- **test:watch**: `npm run test:watch`
- **test:coverage**: `npm run test:coverage`
- **start**: `npm run start`

## Code Conventions
- TypeScript project (v5.2.0)
- Uses Vitest for testing (v0.34.0)
- Uses tsup as build tool (v8.0.0)
- Uses npm as package manager
- Runs on Node.js
- Source code organised under src/
- Tests in __tests__/
- Dual ESM+CJS output via tsup
